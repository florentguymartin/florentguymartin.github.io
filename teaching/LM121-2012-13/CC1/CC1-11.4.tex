\noindent
\textbf{Exercice 2:}\\
\begin{enumerate}
\item 
Déterminer les nombres complexes $w$ tels que 
$w^2 = -7-24i$.\\ 
Pour information, 
$|-7-24i| = \sqrt{ 7^2 +24^2} = \sqrt{ 49 + 576}=\sqrt{625} = 25$.
\item 
En déduire la forme cartésienne des éléments de 
\[ \mathscr{S} = \{ z\in \mathbb{C} \ \big| \ z^4 = -7-24i \}\]
 puis représenter les points 
de $\mathscr{S}$ dans le plan.

\end{enumerate}
