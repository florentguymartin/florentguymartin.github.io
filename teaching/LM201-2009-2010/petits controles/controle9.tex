{\Large Exercice 1 } : \\
\vspace{5mm}\\
Soit $y : \mathbb{R} \rightarrow \mathbb{R} $ une fonction deux fois d{\'e}rivable
solution de l'{\'e}quation diff{\'e}rentielle : 
$$y''(x) + 2y'(x) + 5y(x) = 5x³+6x^2+6x$$
Montrer que $y(x) \xrightarrow[x \to +\infty]{} +\infty$.\\
\vspace{10mm}\\

{ \large Exercice 2 }: \\
\vspace{5mm}

Soit $y : \mathbb{R} \rightarrow \mathbb{R} $ une fonction deux fois d{\'e}rivable
solution de l'{\'e}quation diff{\'e}rentielle : 
$$y''(x) + 2y'(x) + 5y(x) = 10\cos(x)$$
Monrer que $y$ n'a pas de limite en $+ \infty$.
